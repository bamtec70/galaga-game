# GALAGA — Namco 1981 Style

Browser recreation of **Namco’s Galaga** (1981; Midway in North America), companion to:

- [PAC-MAN](https://bamtec70.github.io/pacman-game/)
- [MS. PAC-MAN](https://bamtec70.github.io/ms-pacman-game/)
- [DIG DUG](https://bamtec70.github.io/dig-dug-game/)
- [DEFENDER](https://bamtec70.github.io/defender-game/)

## Play

**Live:** https://bamtec70.github.io/galaga-game/

Or open `index.html` locally:

```powershell
start index.html
```

## Mission

Pilot your **fighter** at the bottom of the screen. Destroy the Galaga forces as they fly into formation and dive-attack. Survive as long as you can — stages never truly end.

## Classic systems

- **Convoy entry** into a swaying formation  
- **Bees**, **Butterflies**, **Boss Galaga** (2 hits)  
- **Diving attacks** with shots  
- **Tractor beam** capture by Boss Galaga  
- **Rescue** the captive while the boss is diving → **Dual Fighter** (double shot)  
- **Challenging Stages** (3, 7, 11, …) with accuracy / perfect bonus  
- Stage badges, extra life every **20,000**  
- Limited shots on screen (2 single / 4 dual)  
- Touch + keyboard controls  

## Controls

| Input | Action |
|-------|--------|
| **← → / A D** | Move |
| **Space** | Fire |
| **Drag** on screen | Move (touch) |
| **P / Esc** | Pause |
| **M** | Mute |

## Scoring (arcade-style)

| Target | Formation | Diving |
|--------|-----------|--------|
| Bee | 50 | 100 |
| Butterfly | 80 | 160 |
| Boss | 150 | 400+ (escorts) |
| Perfect challenge | | 10,000 |
